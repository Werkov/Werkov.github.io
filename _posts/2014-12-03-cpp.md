===== Initializers =====

{% highlight  cpp %}
int a;             // unitialized integer
int b = int();     // b initialized to zero
int c();           // (sic) declaration of function c returning int

Class a;           // calls default ctor
Class b = Class(); // same as above
Class c();         // again, function declaration
